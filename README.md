# Credit Risk Prediction using Machine Learning

## Project Overview
This project predicts whether a loan applicant is likely to default or not based on financial and demographic information.  
The goal is to help banks and financial institutions make data-driven decisions for loan approvals.

---

## Features
- Data cleaning and handling missing values
- Exploratory Data Analysis (EDA) with visualizations
- Feature encoding for categorical variables
- Classification using Logistic Regression and Decision Tree
- Model evaluation using accuracy and confusion matrix

---

## Dataset
- **Source:** [Kaggle - Loan Prediction Dataset](https://www.kaggle.com/datasets/nileshpatel17/loan-default-prediction)
- **Columns:** Gender, Married, Dependents, Education, Self_Employed, ApplicantIncome, CoapplicantIncome, LoanAmount, Loan_Amount_Term, Credit_History, Property_Area, Loan_Status (for training data)
